数据定义

数据类型

  • 数据对象的技术特性的定义
  • 本身不占用内存空间
  • 可以是系统预定义或者用户自定义

数据对象（变量）

  • 内存变量
  • 指定为某一特定数据类型

数据类型和数据对象是单独声明的，数据对象则参照数据类型定义。在程序中利用 TYPES 语句声明数据类型，用 DATA 语句定义变量。

赋值

MOVE 语句

这两种形式是完全等价的，其中源字段可以是变量，也可以是常量、文字等其他数据对象，其值在赋值前后保持不变。赋值后目标变量的值是否与源数据对象完全一致，则取决于二者的类型、长度等技术属性。对于相同基本类型声明的数据对象，如果二者长度不同，需要考虑赋值过程中的对齐方式；对于不同类型数据对象赋值过程，要遵守下列基本原则:

  • 如果源对象和目标对象完全兼容，即二者所有技术属性完全一致（数据类型、长度及小数位等)，则赋值过程中从源对象到目标对象赋值结果保持不变。
  • 如果源对象和目标对象不完全兼容但可以进行类型转换，系统自动按照固定的转换规则进行处理，这个附加过程将耗费一定的系统运行时间。各种类型之间的具体转换规则参考类型转换规则表。
  • 如果源对象和目标对象既不完全兼容也不能进行类型转换，则不能成功完成赋值操作。系统可能在语法检查或编译过程中给出错误提示，否则将出现运行时错误。

结构体赋值

相似的结构体之间进行赋值

使用 MOVE-GORRESPONDING 语句在两个结构体中的所有同名组件间赋值，而目标结构体中其他字段的值不变。

指针

类型转换

在不同类型的数据对象之间赋值，会自动进行类型的转换；

转换过程遵照固定的规则，如 C 类型数据赋值给 N 类型，只有数字字符被传递，其他忽略；

C 不能直接赋值给 I，需要 C ->N -> I；

可以使用 MOVE TO 或 WRITE TO 语句。

日期时间运算

有时也需要对时间和日期类型进行简单计算。这种类型实质上属于字符类型，但在进行计算时，系统会将其字段值首先转换成数值运算类型，因而系统会自动根据这两种特定的数据类型进行合理的增减。例如将日期20041231加1，得到的结果应该是20050101，而不会是20041232,对于时间也是如此，时间相加减后赋给I类型数据的结果是其秒数的差异。
